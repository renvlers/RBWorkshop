# 🏗️ 红建工坊 RBWorkshop

**红建工坊** 是一个 Minecraft 原版小游戏地图，结合了**建筑速建**与**红石速建**两种核心玩法。玩家需要在紧张的时间限制内，展现精湛的建筑美学或缜密的红石逻辑，争取获得最高分。


## 🎮 游戏概览

* **游戏类型**: 原版小游戏
* **核心玩法**: 建筑速建 + 红石速建
* **胜利条件**: 在规定轮次结束后，总分最高的队伍获胜。
* **场地规格**: 每个玩家拥有 `35x35x30` 的独立创作空间。


## 👥 队伍与机制

### 队伍划分

游戏支持以下 6 种颜色的队伍：

* 🔴 **红队**
* 🟠 **橙队**
* 🟡 **黄队**
* 🟢 **绿队**
* 🔵 **蓝队**
* 🟣 **紫队**

> **注意**: 游戏仅启用有玩家加入的队伍，且至少需要 **2 个队伍** 才能开始游戏。

### 轮次系统

* **混合模式**: 游戏实行一局多轮制。
* **自定义轮数**: 玩家可调节建筑轮数和红石轮数（各自上限 5 轮，总和上限 10 轮）。
* **随机抽取**: 游戏开始后，若两种模式均有剩余轮次，系统将**随机决定**当前轮次是“建筑”还是“红石”。


## 🔨 玩法详解：建筑速建

在该模式下，比拼的是创意与手速。

### 流程

1. **选题**: 每轮开始随机给出 5 个主题，玩家投票选出本轮主题。
2. **建造**: 限时 **10 分钟** 完成主题建筑。
3. **评分**: 所有玩家互相给建筑打分（互评），建造者得分为非本队玩家评分的算术平均值。

### 评分标准

每个队伍有 **20 秒** 打分时间，共有五个评价等级：

| 等级 | 分数 | 描述 |
| --- | --- | --- |
| **夯** | **100** | 完美，无可挑剔 |
| **顶级** | **80** | 优秀的作品 |
| **人上人** | **60** | 还可以，中规中矩 |
| **NPC** | **40** | 平平无奇 |
| **拉完了** | **20** | 需要努力 |

### 辅助工具

玩家 HUD 的最后一格（第 9 格）锁定为**右键检测器**：

* **操作**: 手持打开菜单，右键切换选项，丢弃键（通常是 Q）确认选择。
* **功能**:
* 🔄 更换地皮材质
* 🖌️ 获取生物群系刷子（右键目标位置修改群系）
* ☀️ 更改个人时间/天气
* 👤 获取玩家头颅

## 🔌 玩法详解：红石速建

在该模式下，比拼的是逻辑与准确性。

### 流程

1. **选题**: 每轮开始随机给出 5 个红石问题，玩家投票确定本轮题目。
2. **解题**: 系统提供输入形式和预期输出形式。玩家需在 **10 分钟** 内建造红石装置。
3. **测试**: 系统自动进行黑盒测试。

### 评分与测试机制

* **测试样例**: 系统随机抽取 **4 个测试样例**（对玩家保密）。
* **分值分配**: 每个样例 **25 分**，满分 100 分。
* **性能限制**: 每个测试样例的运行判定时间限定在 **5 秒** 内。
* **判定**: 装置需在规定时间内，根据输入产生正确的预期输出。

### 辅助工具

玩家 HUD 的最后一格（第 9 格）锁定为**右键检测器**：

* **操作**: 手持右键点击。
* **功能**: 打开对话框，查看问题描述、输入形式、预期输出形式及具体的时间限制。

## 🏆 结算

* 每轮满分均为 **100 分**。
* 游戏结束时，汇总所有轮次的得分进行最终排名。
